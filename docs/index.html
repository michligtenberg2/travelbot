📡 Travelbot – Jouw AI-reisgenoot

Versie: v0.1

🔧 Wat is Travelbot?

Travelbot is een AI-reisgenoot voor in de auto. Hij praat elke ~15 minuten met een Amsterdams accent over je omgeving, gebaseerd op GPS-gegevens, Wikipedia en een taalmodel. Hij draait op een oude Android-telefoon.

🚀 Installatie

  1. Installeer Python 3.11+ en een Android-telefoon met versie 8 of hoger.
  2. Kloon de repo en zet je OPENAI_API_KEY in je omgeving.
  3. Start de backend met python backend/app.py.
  4. Open de Android map in Android Studio en bouw de app.
  5. Installeer de APK op je toestel en vul het adres van je backend in.
